Предварительный просмотр книги

bQella позволяет посмотреть результат импорта еще до импорта

Для этого идем Проект-> Импорт-> Книга. Замечаем такое окно:

Выбираем какую-нибудь книгу. Выставляем параметры книги (тег главы и тд):

Жмем "Предпросмотр"

Появляется окно:

В нем можно заметить два поля текста. В одном из них весь текст книги, в другом только отдельная глава

Можно посмотреть и код книги, для этого выбираем вкладку "HTML код"

Смотреть на одну главу не интересно, поэтому переключим на какую-то другую

Вот так просто можно посмотреть, как будет выглядеть импортированная книга
