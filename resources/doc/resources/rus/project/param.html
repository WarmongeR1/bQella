Параметры проекта

bQella позволяет установить множество параметров проекта.

Для это необходимо зайти в меню Проект->Свойства проекта

image

Нажав на который, заметим такую картину

image

Такое же окно можно заметить при создании проекта

Если случайно при создании ошиблись и поставили не те настройки, то в этом окне можно изменить параметры, и при экспорте они будут учтены
