Резкалка файлов

bQella позволяет разрезать исходный файл по частям, а затем уже импортировать.

Такая необходимость может возникнуть, если библиотека пользователя хранится в одном файле, а редактировать надо только малую часть

Для этого идем Дополнительные -> Разбить файл или жмем CTRL+ALT+1

Видим диалог:

Какие-то две вкладки. Одна "Вручную", другая "авто".

Пожалуй, надо выбрать файл, выбираем

Файл загружен и в двух текстовых полях что-то появилось.

Как можно заметить, одно текстовое поле поддается редактированию. Это поле необходимо для создание специфического тега для разделения по частям. Или же для мелких правок

А второе для просмотра в привычном глазу формате

Можно отредактировать файл и сохранить изменения

Будем использовать в качестве тега разделения тег главы. Переходим на вкладку "авто"

Здесь уже все введено за нас. Можно запускать. А сначала посмотрим, куда же все это будет резаться. А резаться будет в папку где файл лежит, или куда укажем
