Добавить книгу

Чтобы добавить книгу достаточно нажать на какую-нибудь другую книгу или на ___Instrucion правой клавишей

Можно бует увидет выпадающиее меню:

image

Прочитав которое можно заметить, что есть пункт "Добавить книгу"

Нажав на который мы увидим следующую картину

image

Нам представлены пункты:

  • Полное название
  • Количество глав
  • Короткое название

      • Заполнив их:

      image
