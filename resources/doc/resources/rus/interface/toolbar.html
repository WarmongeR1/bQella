Панель инструментов

Чтобы не приходилось постоянно использовать меню, наиболее часто используемые пункты меню вынесены в панель инструментов.

На данный момент панель инструментов состоит из следующих кнопокф

  • Проект - содержит функции о работе с проектом(модулем)
  • Страницы - позволяет переключаться между вкладками
  • Редактирование - содержит кнопки редактирования текста
  • Дополнительные - в этой панели инструментов содержатся кнопки расположения текста, цвета и прочих функций
