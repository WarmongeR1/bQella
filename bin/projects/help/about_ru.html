Известные значимые ошибки:

- При создании нового проекта или смене текущего, база данных источников не меняется.

Решение: Перезапустить приложение (если собираетесь использовать источники, иначе ошибка вас не коснётся)

Известные незначимые ошибки:

- Не экспортирует только что сделанные изменения в файле.

Решение: Переключиться на какой-нибудь другой файл в списке и затем экспортировать.

- При первом запуске программа закрывается.

Решение: Запустить еще раз. (Не сразу создается файл настроек)
