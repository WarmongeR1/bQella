    bQella.

    bQella - программа для создание модулей для "Цитаты из Библии" без использования тегов.
    Данная разработка позволяет не тратить время на ввод тегов и сконцентрироваться только на тексте.

    Текущая версия

    Ветка разработки

    Начиная с версии 0.1 было решено сделать две ветки разработки - стабильная и ветка разработки(next)

    Ветка разработки почти не тестируется. Но содержит "вкусняшки" и все новые функции сначала выходят в этой ветке

    По мере выхода версий все шлифуется и в стабильную версию уже входят побывавшие в "бою" функции.

    Разработка

    На данном этапе развития программы программа позволяет импортировать простые существующие модули, экспортировать созданные модули, а также производить редактирование модулей без знания тегов.

    Возможности

    • Создание модуля с нуля
    • Использование множества кодировок для импорта.
    • Все опции для создания модулей цитаты.
