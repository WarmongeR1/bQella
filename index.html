    bQella.

    bQella - программа для создание модулей для "Цитаты из Библии" без использования тегов. Данная разработка позволяет не тратить время на ввод тегов и сконцентрироваться только на тексте.

    Текущая версия

    Так как программа пишется на Qt(C++), то программа кроссплатформенна: Свежая сборка под windows: Свежая сборка под linux(source):

    Разработка

    На данном этапе развития программы (pre-alpha6) программа позволяет импортировать простые существующие модули, экспортировать созданные модули, а также производить редактирование модулей без знания тегов.

    Возможности

    • Создание модуля с нуля
    • Использование множества кодировок для импорта.
    • Все опции для создания модулей цитаты.
